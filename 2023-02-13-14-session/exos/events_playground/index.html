Réagir à la soumission d'un formulaire

Lorsqu'il est soumis, un formulaire a un comportement par défaut. Les données seront transmises au script mentionné dans l'attribut 

action
selon la méthode HTTP de l'attribut 
method

Ecrivez le code Javascript qui empêchera la soumission du formulaire si :

  • Le nom n'est pas renseigné ou est d'une longueur inférieure à 2 caractères
  • L'age n'est pas renseigné ou est inférieur à 18
